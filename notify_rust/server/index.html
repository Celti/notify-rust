notify_rust

Module notify_rust::server [−] [src]

Experimental server taking the place of your Desktop Environments Notification Server.

This is not nearly meant for anything but testing, as it only prints notifications to stdout. It does not respond properly either yet.

This server will not replace an already running notification server.
